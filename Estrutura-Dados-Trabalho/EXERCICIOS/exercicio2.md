# Exercício 2
##Por que a busca binária é considerada mais eficiente que a busca linear?

# Resposta
Porque ela reduz o número de elementos pela metade a cada passo.
Diferente da busca linear, a busca binária não percorre toda a lista.  
Ela divide o vetor ao meio e elimina metade dos elementos a cada comparação, diminuindo rapidamente o número de verificações.
