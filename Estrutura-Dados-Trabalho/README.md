### Descrição
Este projeto implementa dois algoritmos de busca em vetores de números inteiros: busca linear e busca binária.  
O programa permite ao usuário procurar um elemento no vetor utilizando um dos algoritmos ou comparar o desempenho entre ambos, mostrando o número de comparações realizadas em cada busca.

### Funcionalidades
- Buscar um número utilizando busca linear
- Buscar um número utilizando busca binária
- Comparar os dois algoritmos de busca
- Exibir a posição do elemento encontrado
- Exibir o número de comparações realizadas

### Linguagem utilizada
C

### Como executar

Compilar o programa:
gcc main.c funcoes.c -o programa

Executar:
./programa
