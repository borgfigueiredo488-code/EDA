# Exercício 6
##Qual é o comportamento da busca binária quando o número não está no vetor?

# Resposta
Ela continua dividindo o vetor até não restarem elementos e então informa que não encontrou.
Mesmo não encontrando o valor, a busca binária segue o processo de divisão até que o intervalo de busca fique inválido (inicio > fim), encerrando o algoritmo.
